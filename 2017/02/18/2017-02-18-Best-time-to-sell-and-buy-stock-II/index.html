Say you have an array for which the ith element is the price of a given stock on day i.

Design an algorithm to find the maximum profit. You may complete as many transactions as you like (ie, buy one and sell one share of the stock multiple times). However, you may not engage in multiple transactions at the same time (ie, you must sell the stock before you buy again).

Solution 1:

Because we can complete as many transactions as we like, buy and sell one stock in one day, so we can get all difference between two numbers. Set a variable profit, traversal the stock prices array, keep add prices[i+1] - prices[i].

Code

Solution 2:

(greedy)
Find each upward interval, use variable index keep the lowest price in the interval, and if prices start to go down, prices[i] < prices[i-1], add this interval’s profit to variable profit.
