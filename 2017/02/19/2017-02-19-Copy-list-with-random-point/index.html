138. Copy List with Random Pointer

By Qi Fu

A linked list is given such that each node contains an additional random pointer which could point to any node in the list or null.

Return a deep copy of the list.

Solution 1:

Divide this problem to three part.

First Copy all node in the list and connect them to the original list node, like 1->2->3 to 1->1'->2->2'->3->3'.

Seconde Connect the copyed node to their random node, like if node 1 random connect to 3, so we connect copyed node 1' to copyed node 3'.

Third Extract the copy list and restore the original list.
